# Polynomial Regression Business Analytics App

This project demonstrates an end-to-end, production-style machine learning
application using polynomial regression for business decision-making.

## Features
- Polynomial & Ridge regression
- Auto degree selection (cross-validation)
- Confidence intervals & uncertainty
- Spend optimization (risk-aware)
- Model comparison dashboard
- Residual drift anomaly detection
- Retraining signals

## Run
pip install -r requirements.txt  
streamlit run app.py
